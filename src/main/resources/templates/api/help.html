微帮Api文档说明

1.编码格式统一使用UTF-8

2.接口请求头配置(错误的请求头会返回服务器异常):application/json;charset=utf-8

3.请求方式以Api文档中标记的方式为准

4.接口访问说明

前端接口直接使用路径+接口名可访问
web页面接口使用路径+接口名+.web访问
登录会返回token，在响应头中，使用token字段返回
业务接口均需要携带userId与token，使用token键上送，格式：userId|token，否则会返回token校验失败的错误

5.接口中密码、身份证认证的证件号码、姓名均需要使用RSA加密后传输，客户端使用服务端约定好的公钥进行加密
